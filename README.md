# Template for NSU

## Title slide

Use pre formated title slide.

Example:

```typst
#title-slide(
  title: [On The Explosion of Large Death Stars],
  author: [Luke Skywalker, Ph.D.],
  lead: [Master Yoda],
  date: [May 25, 1977] 
)
```

## Enums and Lists

You can use lists and enums as usual.
Enums and lissts would have MMD nsu colors.
